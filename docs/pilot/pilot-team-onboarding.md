# Pilot Team Onboarding

## How to reach BreadMind
- `@BreadMind` in any pilot channel to ask a question.
- DM `@BreadMind` for private questions (scoped to your project only).

## What to ask
- "Last year's payment module memory leak — how did we fix it?"
- "CI pipeline extension steps?"
- "New dev environment setup procedure?"

## How answers look
Answer body
📎 Sources: <slack permalink> · <confluence url>
Confidence: 🟢 / 🟡 / 🔴
Buttons: [👍] [👎] [🔖 Save to org KB]

## When to press what
- 👍 → answer was accurate; boosts retrieval weight.
- 👎 → answer was wrong; flags cited KB for lead review.
- 🔖 → fast-track to promotion queue.

## Confidence icons
- 🟢 high — cited sources are strong.
- 🟡 medium — partial evidence, verify before acting.
- 🔴 low — evidence missing; treat as draft only.

## Blocked topics
HR / salary / M&A / security incidents / PII are blocked by design. You will
see: "민감 카테고리로 분류되어 답변이 제한됩니다. 담당자: HR 팀 (#hr-inquiry)".
